# PokeDiary

PokeDiary is your ultimate companion for all things Pokémon! This comprehensive Pokédex app provides detailed information on every Pokémon including stats. Easily track and organize your Pokémon collection with intuitive features. Perfect for trainers of all ages, Poke Diary is your go-to app for becoming a Pokémon master!

## Key Features

- Comprehensive Pokémon List: Access detailed information on every Pokémon from all generations.
- Detailed Pokémon Profiles:
  - Type: View the primary and secondary types of each Pokémon.
  - Shiny Sprites: Check out shiny versions of your favorite Pokémon.
  - Height and Weight: Find physical attributes for each Pokémon.
  - Stats: Analyze base stats, including HP, Attack, Defense, Special Attack, Special Defense, and Speed.
- User-Friendly Interface: Enjoy a seamless and intuitive user experience for all ages.

## Libraries used

- Jetpack Compose:
  - A modern toolkit for building native Android UIs with a declarative approach.
  - Simplifies and accelerates UI development.
- Dagger Hilt:
  - A dependency injection library for Android.
  - Reduces the boilerplate of manual dependency injection and integrates seamlessly with the Android framework.
- Retrofit:
  - A type-safe HTTP client for Android and Java.
  - Simplifies network calls by converting APIs into Java interfaces.
- Coil:
  - An image loading library for Android.
  - Designed for simplicity and performance, leveraging Kotlin coroutines.
- Palette:
  - Extracts prominent colors from images.
  - Useful for creating visually consistent UIs based on image colors.
- Lottie Animation:
  - Renders animations in real-time from Adobe After Effects.
  - Allows for high-quality, scalable animations in apps with minimal effort.
